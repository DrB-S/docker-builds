# NanoPlot container

Main tool : [NanoPlot](https://github.com/wdecoster/NanoPlot)

Additional tools:
- matplotlib

Full documentation: https://github.com/wdecoster/NanoPlot

> Plotting tool for long read sequencing data and alignments.

# Example Usage

```
NanoPlot --barcoded \
  --summary sequencing_summary \
  --threads 8 \
  --outdir nanoplot \
  --raw
```
